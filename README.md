# MARAGI RabbitMQ Container

Broker definitions and Docker launch script. 

# Instructions

1. Install Docker
2. Run rabbit.sh
3. Log into RabbitMQ instance and apply JSON definitions file
4. Start other [MARAGI services](https://github.com/topics/maragi)
